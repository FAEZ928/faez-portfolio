Interactive Photo Gallery

The Interactive Photo Gallery project was created to display images in a visually appealing and interactive format, allowing users to explore various themes effortlessly.

The gallery uses a grid layout with thumbnail images that animate on hover, inviting interaction. Clicking a thumbnail opens a larger view, and users can filter images by theme using JavaScript for a seamless browsing experience.

Captions and descriptions provide context, while CSS ensures responsiveness across devices. Images are optimized for quick loading, enhancing user experience. This project highlights my skills in HTML, CSS, and JavaScript, showcasing my ability to design interactive and performance-focused web applications.

Key Features:

  • Grid layout displaying thumbnails for different collections or themes.
  • Hover animations to enhance visual appeal and user interaction.
  • Captions and descriptions for each image to provide context.
  • Responsive design adapting to various screen sizes.
